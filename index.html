O que eu construí

🌱

Easy Food

Plataforma full-stack alinhada ao ODS 2 (Fome Zero), conectando pequenos produtores rurais a consumidores com foco em sustentabilidade e experiência do usuário.

ASP.NET Core C# React TypeScript SQL Server JWT Material UI

🔧 Backend: API REST com ASP.NET Core (C#), autenticação JWT, banco SQL Server, testes com MSTest e documentação no Postman.

🎨 Frontend: React + TypeScript com componentes Material UI, integração via Axios e deploy na Vercel.

📋 Gestão: Metodologia ágil com Trello, comunicação no Discord e versionamento no GitHub.

🛒

VendasApp

PDV para pequenos comércios instalável como PWA — leitor de código de barras, baixa automática de estoque, controle de papéis e funcionamento offline-first com sincronização ao reconectar.

Next.js 16 TypeScript Supabase PostgreSQL PWA Electron Tailwind CSS

🔧 Backend: Next.js App Router com Server Actions, Supabase (PostgreSQL + Auth + RLS), validação com Zod e relatório diário por email via Vercel Cron + SMTP.

🎨 Frontend: Interface responsiva com Base UI + Tailwind v4, dashboard de KPIs, gráficos Recharts e sidebar adaptável desktop/mobile.

📴 Offline-first: Catálogo cacheado em IndexedDB (Dexie), fila de vendas local com replay idempotente via client_uuid e recibo térmico 80mm imprimível mesmo sem conexão.

💻 Desktop: App Windows instalável (.exe) via Electron — funciona como PDV fixo sem depender do navegador.
